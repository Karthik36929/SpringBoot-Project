## How to Run

### Prerequisites
- Java 17 or higher
- Maven 3.6+

### Build the project
```bash
mvn clean install
```

### Run the application
```bash
mvn spring-boot:run
```

Or run the JAR file:
```bash
java -jar target/greeting-api-0.0.1-SNAPSHOT.jar
```

## API Endpoints

The application runs on `http://localhost:8080`

### Available Endpoints:
- **GET** `/api/greet`
- **GET** `/api/greet/{name}`
- **GET** `/api/greet/custom`
- **POST** `/api/greet`

## Testing

Run tests with:
```bash
mvn test
```
